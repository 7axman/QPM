Why QPM?

⚡️

100% Native Swift

Built directly on Apple's Network framework. Blazingly fast, uses almost zero RAM, and looks beautiful in your Menu Bar.

🔌

Zero Python

No pip, no virtual environments, no broken terminal scripts. Just install it and it runs completely autonomously.

🔄

Auto-Detects

Automatically scans the system to find your active VPN interface. If you disconnect, it instantly detects it.

Installation & Setup

Install securely using Homebrew to bypass Apple's developer quarantine restrictions:

brew install 7axman/qpm/qpm

Prerequisites: qBittorrent Setup

  • Open qBittorrent and go to Preferences > WebUI.
  • Check the box for "Web User Interface (Remote control)".
  • Set a Username and Password in the Authentication section.
  • Open QPM from your Applications folder and enter those credentials into the popover menu!
